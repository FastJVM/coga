# Demo project

A local project scaffolded for smoke-testing the Relay CLI without
touching real code. Tasks created here won't interact with any real
codebase — they're for exercising `create`, `status`, `launch`,
`step`, `panic`, and `feed` end-to-end.

Delete this project (remove from `relay.toml` and `relay.local.toml`)
once you have a real project configured.
